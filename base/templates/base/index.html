01. À propos de moi

Qui suis-je ?

Je suis Mohamed Salmi, développeur Full Stack et Data Analyste basé au Maroc. Passionné par la technologie depuis plus d'une décennie, j'accompagne entreprises et startups dans leur transformation numérique en concevant des applications web modernes, des APIs robustes et des pipelines de données efficaces.

J'interviens également comme formateur en développement web et data science, et je suis à l'origine de tanaoneline.com, un service de création de sites web professionnels pour commerces locaux au Maroc.

  • Python / Django / DRF
  • Flutter (Mobile)
  • Data Analysis & BI
  • REST API Design
  • MySQL / PostgreSQL
  • Git & DevOps
  • JavaScript / Bootstrap
  • Linux / GCP
02. Ce que je fais

Mes services

Développement Web Full Stack

Conception et développement d'applications web complètes avec Django, Python et des frontends modernes. De la maquette au déploiement.

API REST & Intégrations

Création d'APIs RESTful sécurisées avec Django REST Framework pour alimenter vos applications mobiles, SPA ou services tiers.

Data Analyse & Dashboards

Exploration et visualisation de vos données avec Python (Pandas, NumPy, Matplotlib). Tableaux de bord interactifs pour vos décisions.

Applications Mobiles

Formation & Coaching
